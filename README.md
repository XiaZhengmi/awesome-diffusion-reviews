# 精选扩散模型综述论文 Awesome Review Papers on Diffusion Models

如果你想开始学习 **扩散模型的基础知识**，这些论文真的非常棒！  
即使你觉得扩散模型难以理解，这些综述论文都用相对简单的语言解释了复杂概念，是入门与进阶的绝佳资源。  
这里整理了扩散模型在 **机器学习**、**计算机视觉** 和 **生成式 AI** 领域的高质量论文与学习资源。
